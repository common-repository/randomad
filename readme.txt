== Description ==
 
You can show ads random by this plugin.
You can also specify the categories of ads to display.
 
設定された広告をランダムに表示するプラグインです。
カテゴリーで表示させる広告を指定することもできます。
 
== Installation ==
 
You can install this plugin directly from your WordPress dashboard:
 
 1. Go to the *Plugins* menu and click *Add New*.
 2. Search for *RandomAd*.
 3. Click *Install Now* next to the *RandomAd* plugin.
 4. Activate the plugin.
